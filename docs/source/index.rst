Fittrix Api 문서에 오신 것을 환영합니다.
===================================

본 문서는 (주)피트릭스에 제공하는 API 가이드입니다.
자사에서 제공하는 API는 자사의 바디미터에서 측정된 신체 데이터를 확인하기 위한 용도로 사용됩니다.
사용자는 자사의 바디미터를 통해서 자신의 신체 정보를 측정할 수 있으며 API를 호출하는 업체에서는 귀사가 소유한 바디미터를 통해서 측정한 데이터에 대해서 정보를 취득할 수 있습니다.
현재 제공하는 API는 신체 측정 데이터를 기반으로 하며 자사의 `관리자 페이지 <https://manage.fittrix.io/>`_를 통해서 확인할 수 있는 데이터를 기반으로 하고 있습니다.

.. note::

   해당 문서는 기능이 추가되거나 변경될 경우 주기적으로 업데이트될 예정입니다.
